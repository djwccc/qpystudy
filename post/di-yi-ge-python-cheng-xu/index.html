2021-02-19

print("Hello, world!")

打开python自带的IDE，写入

print("Hello, world!")

然后，运行~

现在可以看到，你成功运行了第一行代码！

同时，你还可以选择创建一个python文件进行编写
在python自带的IDE上方Files，点击New File

你就可以创建一个python文件了

python文件的后缀名都基本上是.py
有些是其他的后缀名
